CSS Basics
What is CSS?

CSS stands for Cascading Style Sheets. It is not actually a programming language but it is infact a style sheet language. In order to use CSS, you must first learn HTML as CSS is used to style HTML. CSS selects HTML elemnts to individually style.


To apply the CSS to our HTML elements, it is crucial that the files are both linked. An easy way to do this is to add the following short line of code into the head of our index.html file:




  <link href="styles/style.css" rel="stylesheet">

How to write CSS

CSS syntax is very simple but there are a few things to know. To start, we must think about which HTML element we want to select. For elements we would just use the word of the element.

For example, to select a <p></p> tag we would write the syntax:

  p {
      
  }

Anything we put into this selector would style that p tag.

For class attributes, we use a period (.) before writing out the value of the class and for id attributes we use a hash (#) before the value of the id.

Selecting the class of <header class="header"> would look like:

.header {
      
  }

Selecting the id of <header id="header"> would look like:

#header {
      
  }
Basic Styling Methods

Now we know how to select a HTML element but how do we add styles?

There are many different ways to style HTML such as thinking about the font, color and even size of an element.

The basic syntax is to decide what you want to style and then write the style name for that. After the style name we add a colon (:) and then the specific change we want. Then we add a semicolon (;) on the end.

For example, if we wanted to change the color of a paragraph to red. First, we would select the p and add cursive brackets. Inside these brackets we would write our style name, in this case "color" and then put a colon. Then we would write "red" and add a semicolon on the end.

The syntax would look like this:

  p {
   color: red;
  }

Basic Styles:

  • font-family = The typeface of our element.
  • font-size = The size of the font written in pixels (i.e. 20px).
  • font-weight = the thickness of our font (i.e bold, normal, lighter).
  • color = To change the color.
  • opacity = Transparency of element.
  • background-image = To add a background image, use url () for this.
  • text-align = To align text along the page (i.e. left, center, right).

There are many more styles not included on this document so feel free to do your own research.

The Box Model

The box model is about how elements are positioned and displayed on a website. The computer has to be told which elements positions need to be altered.

The properties of a box model include:

  • Width and height = The width and height of the content area.
  • Padding = The amount of space between the content area and the border.
  • Border = The thickness and style of the border surrounding the content.
  • Margin = The amount of space between the border and the outside edge of the element.

We can also only add a property to one side of an element using: top, right, bottom or left

For example, if we wanted to add a margin of 20 pixels to the left of a h1 element, we could write:

 h1 {
  margin-left: 20px;
 }
Positioning

There are ways to position elements on the page. One is by using the position style. These include:

  • position
  • display
  • z-index
  • float
  • clear

The position property can take 1 of 4 values:

  • Static - The default value (does not need to bbe specified).
  • Relative - Allows you to position an element relative to its default position.
  • Absolute - The element will be positioned relative to its closest positioned parent elements.
  • Fixed - The element will be fixed on the page. Scrolling will not make it move either.

z-index - When elements overlap eachother, we can use z-index to make one appear over the other. To do this we add a z-index property and give it a number. The higher number will show infront.

float- This is to move an element as far left or as far right as possible (i.e. left, right)

clear- Using float can cause elements to bump into eachother. To avoid this, we can use clear to make space around the sides of the element. Right clears right side. Left clears the left side. Both clears both sides.
